# Equation Formatting Rules

## Display Equations

- All equations must be wrapped in **double dollar signs**: `$$...$$`
- Never use single dollar signs (`$...$`) for displayed equations
- Each equation should be on its own line with blank lines above and below

## Inline References

- When referencing a variable or symbol within body text, use italics: *v*, *a*, *F*
- For inline mathematical expressions, use `$$...$$` on the same line

## Worked Examples

- Centre all calculations in worked examples
- Show each step on a separate line
- Include units at every step, not just the final answer
- Box or bold the final answer where appropriate

## Unit Formatting

- Always leave a space between the number and the unit: `5 m`, `9.81 m s⁻²`
- Use SI units unless the specification requires otherwise
- Negative exponents preferred over fraction notation for units: `m s⁻²` not `m/s²`
- Unit symbols are not italicised; variable symbols are

## Consistency

- Use the same notation as the exam board's specification and formula sheet
- If the specification uses *v* for velocity, do not switch to *u* without reason
- Maintain consistent use of the equation editor throughout a piece of content
- Subscripts and superscripts must be properly formatted (not plain text)

## Common Physics Equations

When writing equations, follow the exam board's formula booklet exactly. For example:

- Kinematic equations: $$v = u + at$$, $$s = ut + \frac{1}{2}at^2$$
- Newton's second law: $$F = ma$$
- Weight: $$W = mg$$
- Energy: $$E_k = \frac{1}{2}mv^2$$, $$\Delta E_p = mg\Delta h$$

## Greek Letters

- Use proper Greek letter symbols (not word approximations): $$\lambda$$ not "lambda"
- Common physics Greek letters: $$\alpha$$, $$\beta$$, $$\gamma$$, $$\delta$$, $$\theta$$, $$\lambda$$, $$\mu$$, $$\rho$$, $$\sigma$$, $$\omega$$
