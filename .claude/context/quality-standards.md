# Quality Standards

## Content Quality Index (CQI)

All content must score **43/50 minimum**. Each criterion scored 1-5.

### Critical Tier (must score 5/5 each)
1. **Specific** — Correct exam-board terms, command words, definitions; no extraneous content
2. **Accurate** — 100% accurate equations, formulas, facts, figures, worked solutions
3. **Concise** — Effective bullets, minimal sentences, no repetition

### Standard Tier (must score 4/5 minimum each)
4. **Correct** — Spelling/grammar correct; UK English (US for AP); 4-eyes review
5. **Consistent** — Course-specific elements used uniformly throughout
6. **Sensitive** — Global audience; sensitivity filter applied; culturally aware
7. **Structured** — Logical heading hierarchy (H1 > H2 > H3, never skip); alt text; cross-references
8. **Formatted** — Uncluttered layout; bullet points; WCAG AAA target
9. **Appropriate Tone** — Trusted expert, professional but not intimidating, not jokey
10. **Appropriate Pitch** — Age-appropriate; CEFR C1/C2 (B2 for EAL); Hemingway reading level 10-12

## Gold Standard SFMA

SFMAs use three solution-box components in this order:

1. **Model Answer** — Full worked solution achieving all marks
2. **Mark Scheme & Guidance (MS&G)** — Marking points, accepted alternatives, command word guidance
3. **Examiner Tips & Tricks (ET&T)** — Actionable exam advice, common errors, strategic tips

### Rules by Question Type

**MCQ**: Free text explanation only (no MS&G). Structure: bold green final answer statement, explanation of correct answer, optional explanation of incorrect options, optional images (black/grey/white only). No marks shown.

**SQ Non-levelled**: Minimum requirement is Model Answer OR MS&G (or both). Model Answer shows full mark-earning solution. MS&G adds marking points only when Model Answer does not show all pathways. ET&T provides new insight (never repeats the solution).

**SQ Levelled**: Includes level descriptors in MS&G. Model Answer demonstrates top-level response.

### Common Mistakes to Avoid
- Adding MS&G when the Model Answer already shows all mark-earning steps
- Including commentary or explanation inside MS&G
- Forgetting grouping logic (e.g. "Any one from:")
- Using coloured diagrams (only black, grey, white)
- Writing ET&T that repeats the solution or reteaches content
