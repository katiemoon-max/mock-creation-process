# Misconception Bank — {{BOARD}} {{QUALIFICATION}} {{SUBJECT}}

Every MCQ distractor across this paper must correspond to a **named** misconception in this file. New distractors append new misconceptions here. Examiner reports and past-paper mark schemes are the primary source.

## How to use this file

- When drafting an MCQ (`/mock-3-draft`), each distractor (B, C, D...) must cite a misconception below by ID.
- If a distractor doesn't match an existing entry, add a new one with an ID, short name, worked description, and source.
- Examples of sources: "AQA Examiner Report June 2022, Q3", "seeded from /mock-1-research distractor archetype analysis".

## Distractor archetypes (subject-agnostic)

These are baseline patterns. The subject-specific entries live below.

| ID | Archetype | Description |
|----|-----------|-------------|
| A001 | Calculation slip | Student follows correct method but makes an arithmetic or transcription error |
| A002 | Unit confusion | Student uses a related unit incorrectly (e.g. cm vs m, minutes vs seconds) |
| A003 | Reversed definition | Student swaps two related quantities or confuses the direction of a relationship |
| A004 | Formula confusion | Student uses a closely-related but wrong formula |
| A005 | Partial-method | Student stops after partial working |
| A006 | Over-generalisation | Student applies a rule outside its domain of validity |
